O miniTruco é um jogo de Truco feito para dispositivos móveis que possuam J2ME (CLDC) implementando MIDP 1.0 ou superior, que permite o jogo individual ou, se o dispostivo tiver Bluetooth e suportar a JSR-82, jogo multiplayer com até quatro jogadores.

Versões antigas permitiam o jogo multiplayer via GPRS, mas este feature foi abandonado devido à dificuldade de encontrar entre as operadoras o suporte apropriado.

Para maiores informações sobre as classes descritas aqui (diagramas, repositório Subversion, etc.), consulte a documentação do desenvolvedor

O jogo pode ser obtido no chester.blog.br.
